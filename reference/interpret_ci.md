# Interpret confidence interval

Provides plain-language interpretation of confidence interval

## Usage

``` r
interpret_ci(ci, conf_level = 0.95, mu0 = NULL)
```

## Arguments

- ci:

  Numeric vector with lower and upper bounds

- conf_level:

  Confidence level (e.g., 0.95)

- mu0:

  Null hypothesis value (optional)

## Value

Character string with interpretation
