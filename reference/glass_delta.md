# Calculate Glass's delta

Glass's delta uses the population standard deviation (if known) or
treats mu0 as the control group mean

## Usage

``` r
glass_delta(x, mu0, sigma = NULL)
```

## Arguments

- x:

  Numeric vector of observations

- mu0:

  Null hypothesis value

- sigma:

  Population standard deviation (if known)

## Value

Numeric Glass's delta value
